# Super-Safe-Deposit-Box-lego-spike-app
LEGO SPIKE App project that simulates a secure deposit box. This project uses motors, sensors, and the SPIKE app to create a “super safe” lock system, including features like a color-coded unlock mechanism and interactive user feedback. 

# You will need:
1. Lego Spike Prime online editor: https://spike.legoeducation.com/prime/lobby/
2. Lego set and build instruction manual: https://spike.legoeducation.com/prime/models/blt2fa30a2282c8da6c
